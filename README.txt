1. 주차장에서 자리가 추가, 삭제되는 것을 시각적으로 볼 수 있습니다.
2. 원한다면 주차장의 자동차의 위치를 알 수 있습니다.
3. 자동차 번호판에는 숫자 8자리만 테스트했습니다.
4. 삭제 기능 중 존재하지 않는 번호판을 입력하면 삭제는 실행되지 않고 메인화면으로 오게 됩니다.
# 주차장 관리 프로그램

이 프로그램은 주차장에서 차량의 추가 및 삭제를 시각적으로 보여주고, 차량의 위치를 확인할 수 있는 기능을 제공합니다. 8자리 숫자 번호판을 지원하며, 존재하지 않는 번호판을 입력할 경우 삭제 기능이 작동하지 않습니다.

## 소개

주차장 관리 프로그램은 차량의 주차 상태를 관리하고 시각적으로 보여주는 도구입니다. 사용자는 차량을 추가하거나 삭제할 수 있으며, 현재 주차된 차량의 위치를 확인할 수 있습니다.

## 기능

- 주차장에 차량 추가 및 삭제
- 차량의 위치 확인
- 8자리 숫자 번호판 지원
- 존재하지 않는 번호판 입력 시 삭제 기능 비활성화

## 설치 방법

1. **소스 코드 다운로드**: GitHub에서 소스 코드를 클론하거나 ZIP 파일로 다운로드합니다.

   ```bash
   git clone https://github.com/사용자이름/주차장관리프로그램.git

2. 필요한 라이브러리 설치: 프로그램에 필요한 라이브러리를 설치합니다.

# 예시 (필요한 라이브러리 설치)
npm install


## 사용법
1. 프로그램을 실행합니다.
./주차장관리프로그램


2. 메뉴에서 원하는 기능을 선택합니다.
- 차량 추가
- 차량 삭제
- 차량 위치 확인
- 차량 번호판을 입력하고, 프로그램의 안내에 따라 진행합니다.
